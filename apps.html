browser plugins

  • text sanitizer / 文字消毒器
    simple browser plugin
    replacing politically convoluted terms on websites with its correct and accurate counterpart which makes sense. you won't see convoluted terms in your browser anymore!
    inspired as i found myself talking about the 'novel coronavirus pneumonia', an entirely fabricated lexical item obfuscating its origins and what's really going on.
    (2020)

    COVID -> Chinese Virus etc. (english)
    新冠 -> 中囯病毒 etc. (chinese)
    新型コロナ -> 武漢コロナ etc. (japanese)
    optional: write your own

    click the button 'set to default and save' to load the default replacement list.
    otherwise, the list is empty.

  • poetry generator
    written during the a.i. frenzy (2016)
    python command line, no GUI
    [won't be released]
    the programme reviewed some chinese poetry and can compose new pieecs in a click. users may input seed words if wanted to. turing tests was conducted to collect comments and determine reliability. pieces were collect from some poet friends for the comparison test. it is said that the programme is a little dadaist, which i sometimes agree - it is a little bit jumpy but i don't think it is a problem. the results reviewed that the program is on average 70% human, and for some pieces, over 85% human.
    another poetry programme 少女诗人小冰 from 微软小冰 (xiaoice / microsoft xiaobing) went popular in mid-2017. since then, the intention in polishing my programme perished.
